> A Pokémon Go CP Estimation Calculator


## Install

```
$ npm install --save pokemon-cp-calc
```


## Usage

```js
const pokemonCpCalc = require('pokemon-cp-calc');

pokemonCpCalc('unicorns');
//=> 'unicorns & rainbows'
```
